# 微信小程序

### 适用场景

​		小程序是运行在微信上的一个应用，`微信本身的资源就限制了小程序性能的上限`，因此小程序的适用场景很少。

!> 千万不要手上有一把锤子，看什么都像钉子

​	小程序的适用场景：

1.   业务逻辑简单：因为简洁，所以简单；因为简单，所以简洁；随点随用，随关随走

2.   使用频率低

3.   性能要求不高

    ​

### 移动布局技术

*    物理分辨率
*    设备独立像素
*    每英寸包括的像素数： ppi
*    每英寸包括的点数： dpi
*    视网膜屏原理

!> 规定在 iphone6s 的情况下，750 rpx = 750 物理分辨率 = 375 (水平方向的设备独立像素)

---

### 文字

​	文字的大小建议设置为 `px`，避免使用 `rpx`
