# 字符串

### 字符串运算

```
# 加法运算
'alice' + ing => 'aliceing'

# 减法运算
alice - ing => alice-ing

# 除法运算
alice / ing => alice/ing

# 乘法运算
alice * ing => Undefined operation "alice * ing"
```

### 字符串函数

```bash
# 转大写
to-upper-case("hello world") => "HELLO WORLD"

# 转小写
to-lower-case("HELLO WORLD") => "hello world"

# 字符串长度
str-length("hello world") => 11

# 查找字符串索引
str-index("hello world", "wo") => 7

# 在指定位置插入字符
str-insert("hello world", " hello boy", 12) => "hello world hello boy"
```
