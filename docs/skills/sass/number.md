# 数字

### 数字运算

```bash
# 加法运算
5px + 5px  => 10px

# 减法运算
5px - 2 => 3px

# 乘法运算
5px * 2px => 10px*px
3 + 2 * 5px => 13px

# 除法运算
8 / 2  => 8/2
(8 / 2) => 4
(8px / 2px) => 4
(8px / 2) => 4px
```

### 数字函数

```css
# 绝对值
abs(10) => 10
abs(10px) => 10px
abs(-10px) => 10px

# 四舍五入
round(3.2) => 3
round(3.5) => 4

# 进位
ceil(3.1) => 4
ceil(3.5) => 4

# 退位
floor(3.6) => 3

# 转百分数
percentage(650px / 1000px) => 65%

# 最小值
min(1, 2, 3) => 1

# 最大值
max(1, 2, 3) => 3
```
